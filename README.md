# NuxtJS-Barcode-Scanner

Barcode Scanner Example in Nuxt.js based on [Dynamsoft Barcode Reader](https://www.dynamsoft.com/barcode-reader/overview/).

[Online demo](https://nuxtjs-barcode-scanner.netlify.app/)

## How to Test

```bash
npm install
npm run dev
```
